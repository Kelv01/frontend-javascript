# Task 0 - Creating a Student Interface and Rendering a Table

## Description
This project introduces basic TypeScript concepts by creating an interface and rendering a table using Vanilla JavaScript.

## Learning Objectives
- Define and use a TypeScript interface
- Create typed objects and arrays
- Use DOM methods to render HTML elements dynamically
- Configure and run TypeScript with Webpack

## Files
- **main.ts** — contains the TypeScript code for defining the `Student` interface, creating students, and rendering a table.
- **index.html** — entry point that loads the bundled JavaScript output.
- **webpack.config.js** — Webpack configuration for compiling TypeScript.
- **tsconfig.json** — TypeScript compiler configuration.

## Usage
1. Run `npm install` to install dependencies.
2. Build the project with:
   ```bash
   npm run build
